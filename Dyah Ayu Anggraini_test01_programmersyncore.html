Test 1 Pengetahuan Dasar

1. Mengapa setiap bahasa pemmrograman itu terdapat framework, apa kegunaan framework?

Jawaban : framework adalah kumpulan struktur dan aturan yang disediakan untuk membangun aplikasi dengan menyediakan fungsi umum, alat bantu, dan pola desain yang telah teruji.

2. Jelaskan mengenai fungsi dan penggunaan if else, while, while do, switch case, for, foreach?

Jawaban : if else digunakan untuk mengecek kondisi mana yang sesuai dengan yang sudah ditentukan, while digunakan untuk perulangan saat kondisi sesuai atau benar, while do digunakan seperti while namun sebelumnya kode akan dieksekusi minimal satu kali sebelum mengecek kondisi, switch case digunakan untuk memilih salah satu kondisi yang sesuai, for digunakan untuk perulangan dimana sudah ada batasan atau length yang telah ditentukan, foreach digunakan untuk perulangan sebuah objek.

3. Apa itu constructor, method, class, object, variabel, parameter, function, procedure ?

Jawaban : constructor yaitu method di dalam class yang dijalankan saat objek dari class dibuat, method yaitu sebuah fungsi yang berkaitan dengan objek ataupun class, class yaitu sesuatu yang mendefinisikan atribut yang dimiliki oleh objek, object yaitu instance dari sebuah class, variabel yaitu atribut yang dimiliki oleh objek di setiap class untuk menyimpan data, parameter yaitu nilai yang diterima oleh fungsi, function yaitu sebuah code yang digunakan untuk mengerjakan tugas, procedure yaitu perintah yang harus dilakukan secara berurutan.

4. Perbedaan method=”Post” dan “Get” ?

Jawaban : method="Post" digunakan untuk mengupload atau mengirim data, sedangkan method="Get" digunakan untuk mendapatkan atau mengambil data dari server.

5. Jelaskan konsep mengenai web service dan fungsi apa yang digunakan untuk memanggil web service client ?

Jawaban : web service adalah teknologi yang memungkinkan aplikasi untuk berkomunikasi dengan aplikasi lain melalui jaringan. Fungsi yang digunakan untuk memanggil web service client dapat bervariasi tergantung pada bahasa pemrograman yang digunakan, tetapi umumnya fungsi untuk melakukan permintaan HTTP (seperti curl dalam PHP).

6. Jelaskan mengenai MVC dan HMVC !

Jawaban : MVC (Model-View-Controller) merupakan desain arsitektur perangkat lunak yang memisahkan aplikasi menjadi tiga bagian utama yaitu model (untuk representasi data), view (untuk tampilan yang diperlihatkan kepada pengguna), dan controller (untuk mengatur alur logika aplikasi). Sedangkan HMVC (Hierarchical Model-View-Controller) merupakan sebuah varian dari arsitektur MVC yang memungkinkan pemisahan lebih lanjut dengan menempatkan komponen-komponen MVC ke dalam struktur hierarkis.

7. Sebutkan fungi curl() dalam php!

Jawaban : fungsi curl() dalam PHP digunakan untuk mengambil data dari server web, mengirim data ke server, atau berinteraksi dengan API.

8. Bagaimana menampilkan sebuah element array dalam PHP ?

Jawaban : menggunakan indeks elemen tersebut misalnya, jika memiliki array $arr, maka dapat menampilkan elemen pertama dengan menggunakan $arr[0]. Atau jika ingin menampilkan semua element array maka bisa menggunakan perulangan.

9. Perbedaan HTTP dan HTTPS ?

Jawaban : perbedaan HTTP dan HTTPS terletak pada keamanannya. HTTPS menggunakan enkripsi SSL/TLS untuk mengamankan data, sedangkan HTTP tidak. Ini membuat HTTPS lebih aman, terutama untuk transfer data yang sensitif.

10. Jelaskan Apa itu API? sebutkan contohnya?

Jawaban : API (Application Programming Interface) adalah kumpulan aturan dan protokol yang memungkinkan aplikasi untuk berkomunikasi satu sama lain. Contoh API yaitu Google Maps API untuk menampilkan peta dalam aplikasi web.
